📞 TROUBLESHOOTING

Problem: "Docker Desktop is unable to start"
Solution: 
  wsl --unregister docker_data
  Restart Docker Desktop from Start Menu

Problem: "unauthorized" when docker push
Solution:
  docker logout
  docker login -u siddartha6174
  (Re-enter credentials, then retry push)

Problem: Docker build fails with timeout
Solution:
  Restart Docker Desktop
  Check internet connection
  Free up disk space (needs ~3GB)

Problem: Kubernetes pods won't start
Solution:
  kubectl describe pod <pod-name>
  (Check for ImagePullBackOff or other error messages)
