1. Preprocess data:
   python scripts/preprocess_cicids2017.py --input data/raw --out features

2. Train model:
   python scripts/train_autoencoder.py --train features/train_normal.csv \
     --val features/val.csv --outdir model

3. Build container:
   docker build -t ids:latest .

4. Deploy to Kubernetes:
   kubectl apply -f k8s/

See PRODUCTION_SETUP.md or docs/README_DEPLOY.md for detailed instructions.
